AWStats is installed on the \texttt{intdev}\footnote{\texttt{intdev} - 
\texttt{dnratwintdev.dnr.state.ak.us} (204.89.222.43)} server from
packages.  The installed versions include:
\begin{itemize}

  \item AWStats v7.7-1.1.el6
\end{itemize}
